Comment Analyzer is the plugin used for finding the inner meaning of a comment and showing the sentiment icon along with the comment listing.

== Description ==

It shows the sentimental value of a comment. It uses the meaningcloud (www.meaningcloud.com) API for finding the sentimental value of the comment. When acticated the sentimental icon will show in the comments listing.

== Installation ==
1. Upload \"comment_analyzer\" to the \"/wp-content/plugins/\" directory.
2. Activate the plugin through the \"Plugins\" menu in WordPress.
3. Create account and get API Key from https://www.meaningcloud.com/developer/login
